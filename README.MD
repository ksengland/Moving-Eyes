# Moving Eyes

<img>

## Description:
Channel your inner Mona Lisa, or spook your friends with these interactive moving eyeballs. Watch the eyes follow your mouse around the screen. Endless hours of fun.

## Contents:
- eyes.js
- index.html
- styles.css

## Instructions:
**Option 1**
<br>
Click <a href="https://ksengland.github.io/Moving-Eyes/" target="_blank">here</a> to run in your browser
<br><br>
**Option 2**
1. Fork this repository
2. Clone your forked repository
3. Open the repository and drag `index.html` onto your browser
4. Move your mouse around your screen to see the eyes follow you

## Built With:
- JavaScript
- CSS
- HTML
